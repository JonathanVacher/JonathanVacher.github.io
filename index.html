Bayesian Modeling of Motion Perception using Dynamical Stochastic Textures

This paper details the complete formulation of a generative model of dynamic textures intended to probe visual motion perception. We use the model to probe speed perception in humans psychophysically using zoom-like changes in stimulus spatial frequency content. The human data replicates previous findings that relative perceived speed is positively biased by spatial frequency increments. The effect cannot be fully accounted for by previous models, but the current prior acting on the spatio-temporal likelihoods has proved necessary in accounting for the perceptual bias.

IPOL Paper: How to Apply a Filter Defined in the Frequency Domain by a Continuous Function ?

This paper provides a theoretical framework designed to analyse the classical and computationally efficient filtering implementations based on discrete Fourier transforms (DFT). It comes with an online demo and a open source code in C.

NIPS Paper: Texture Synthesis For Probing Motion Perception

This paper presents a dynamic texture model for the neuroscience of vision. It is used in a psychophysic study of speed perception embedded in a Bayesian model. The NIPS (Neural Information Processing System) conference was held from 7 to 12 December 2015 in Montreal and gathers data scientists from many fields.

IPOL Paper: The Heeger & Bergen Pyramid Based Texture Synthesis Algorithm

This paper details precisely the Heeger-Bergen texture synthesis algorithm. It comes with an online demo and a open source code in C.
